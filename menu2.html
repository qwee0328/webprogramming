빅데이터 기술

빅데이터의 분석, 활용을 위한 빅데이터 처리기법은 크게 빅데이터 분석을 위한분석 기술과 데이터를 관리, 분석하기 위한 표현기술로 나눌 수 있다.

빅데이터 분석 기술

분석기법에서 대표적인 방법은 마이닝 기법이다. 마이닝 기법은 다량의 데이터에 숨겨진 패턴과 관계등을 파악해 미래를 전망할 수 있는 정보를 추출하는 기술을 말한다. 최근엔 그림이나 영상같은 비정형 데이터의 증가로 다양한 분석방법이 주목받고 있다.

텍스트 마이닝

텍스트 마이닝(text mining)이랑 대규모의 문서에서 의미있는 정보를 추출하는 것을 말한다. 분석 대상이 비구조적인 문서정보라는 점에서 데이터 마이닝과 차이가 있다. 텍스트 마이닝은 정보검색, 데이터 마이닝, 기계학습, 통계학, 컴퓨터 언어학 등이 결합된 학제적 분야이다. 분석 대상이 형태가 일정하지 않고 다루기 힘든 비정형 데이터이므로 인간의 언어를 컴퓨터가 인식해 처리하는 자연어 처리 방법과 관련이 깊다. 텍스트 마이닝은 텍스트 분석, 문서 마이닝 등으로 부르기도 한다.

오피니언 마이닝

오피니언 마이닝(opinion mining)이란 웹사이트와 소셜 미디어에서 특정 주제애 대한 여론이나 정보글을 수집하고 분석해 결과를 도출하는 것을 말한다. 인물, 이슈 등에 대한 대중들의 의견이나 평가, 감정 등을 분석하는데 사용한다. 사실과 의견을 구분해 추출, 긍정과 부정으로 나누고 그 강도까지 측정이 가능하다. 분석 대상이 텍스트이므로 텍스트 마이닝에서 활용하는 자연어 처리방법과 컴퓨터 언어학을 활용하며 소셜 미디어 시대의 중요한 기술로 자리매김 하고 있다.

웹 마이닝

웹 마이닝(web mining)은 인터넷을 이용하는 과정에서 생성되는 웹 로그정보나 검색어로부터 유용한 정보를 추출하는 웹 대상의 데이터 마이닝을 말한다. 웹 마이닝에선 웹 데이터의 속성이 반정형 혹은 비정형이고, 링크 구조를 형성하고 있기 때문에 별도의 분석기법이 필요하다. 웹 마이닝은 분석 대상에 따라 웹 구조 마이닝(web structure mining), 웹 유시지 마이닝(web usage mining), 웹 콘텐츠 마이닝(web contents mining)으로 구분할 수 있다.

기술분석

기술분석(Descriptive analysis)은 빅데이터 분석의 가장 기본이 되는 분석으로, 주어진 데이터를 요약/집계하여 결과를 도출하는 것이 목표이다. 과거의 데이터를 단순 계산/집계하여 얻어진 사실이므로 분석 결과를 따로 해석하는 과정은 거치지 않는다.

탐색적 분석

탐색적 분석(Exploratory analysis)은 EDA라고 부르기도 하며 시각화(Visualization)이 데이터 분석의 화두로 떠오르면서 주목을 받게된 분석 유형이다. 주요 목표는 여러 변수 간 트랜드나 패턴, 관계를 찾는것으로 그래프를 통한 사실 확인이 주된 작업이다.

추론분석

추론분석(Inferential analysis)은 샘플-모집단 간의 관계를 탐구하는 것이 목적이다. 샘플에서 얻어낸 정보가 모집단에도 적용될 수 있는지를 검토하는 것이며, 샘플의 크기를 무한정으로 늘릴 수 없기 때문에 발생하는 일반적인 문제를 해결하기 위한 분석유형이다.

예측 분석

예측분석(Predictive analysis)은 머신러닝, 의사결정트리 등 다양한 통계적 기법을 사용하여 미래 혹은 발생하지 않은 어떤 사건에 대한 예측을 하는 것이 주요 목표이다. 왜 발생하는지 보다는 정확한 예측을 하는것이 주요 관심사기때문에 알고리즘에 대한 설명력은 약하다.

소셜 네트워크 분석

소셜 네트워크 분석(Social Network Analysis)이란 이벤트 로그 데이터를 작업자, 사회적 관점에서 분석하는 것이다. 작업자는 이벤트 로그의 속성 중에 누가 수행했는지를 나타내는 속성이다. 소셜 네트워크 분석을 위한 방법에는 작업자-액티비티 매트릭스(Resource-Activity matrix), 핸드오버 매트릭스(Handover of work matrix) 등이 있다.

빅데이터 표현기술

빅데이터 분석 기술을 통해 분석된 데이터의 의미와 가치를 시각적으로 표현하기 위한 기술로 대표적으로는 R 언어가 있으며 여러 데이터 시각화 툴도 사용 가능하다.
